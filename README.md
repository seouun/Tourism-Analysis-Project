# Tourism Analysis Project
국민여행조사 기반 관광 데이터 분석 (제 3회 BDA 채용 공모전 제출 프로젝트)

## 프로젝트 소개
- 국민여행조사 데이터를 활용하여 관광 패턴 및 숙박 인프라 입지를 분석
- 데이터 기반 정책 제안 및 지역경제 활성화 전략 모색

## 사용 기술
- Python (pandas, numpy, scikit-learn, matplotlib, seaborn)
- Jupyter Notebook

## 주요 결과
- 고지출 숙박형 지역: 여수 최종 선정
- 여수 내 주요 관광지와 빈집 지역 결합 후 최종 입지 분석
- 여수 내 실제 숙박시설 유치 후 기대효과 도출
  
## 파일 설명
- `제3회 BDA 채용 공모전_P-Valley.ipynb`: 메인 분석 노트북
- `제3회 BDA 채용 공모전.pdf`: 1차 보고서 자료
- `제3회 BDA 채용 공모전(2차 심사 파일)_P-Valley.pdf` : 2차 본선 발표 자료
